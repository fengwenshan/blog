# 前言



## 本主题解决什么问题

很多人学大模型时会遇到三个常见问题：

1. 学到的都是碎片词汇，缺少主线
2. 看了很多原理，却不会写应用代码
3. 会调 API，但不会做 RAG、Agent、工程化和项目包装

这套要解决的，就是从“知道术语”到“能够交付应用”之间的断层。

## 适合你的学习方式

推荐按下面方式学习：

- 先通读模块首页，建立知识地图
- 再逐章敲代码，而不是只看结论
- 每学完一个模块，就做一个最小 Demo
- 学完 RAG、Agent、工程化后，再做完整项目
